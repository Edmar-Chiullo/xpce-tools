*O que é o PCE Tools?*

É um sistema web completo de gestão logística e operacional para centros de distribuição, construído com Next.js + Firebase. Ele unifica operações de portaria, recebimento, conferência, picking, endereçamento e quarentena de produtos em uma única plataforma — com versões Desktop (gestão) e Mobile (operadores de chão de fábrica).
Módulos identificados:

- Portaria — Controle de entrada/saída de caminhões com emissão de canhotos
- CPD — Processamento de notas fiscais com timeline por motorista/transportadora
- Recebimento — Gestão completa de cargas e conferência
- PCE Tools (Atividades) — Tarefas em tempo real: picking rotativo, quarentena fracionada, validação produto x endereço, rotativo de aéreo
- Cadastro — Gestão de usuários com permissões
- Dashboard — Painel analítico
- Geração de Códigos — Impressão de etiquetas e códigos de barra (DUN/EAN)
- Importação/Exportação — Leitura de planilhas XLSX
